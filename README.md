# Oldpainless CRT Shaders for WRGB WOLED Panels (v32.0)

These presets are engineered strictly for 4K self-emissive WRGB WOLED panels (tested on the Sony KD-55A85), bypassing sterile digital computer monitor rules to capture true analog light-field cavity rebound mechanics.

## Key Optimizations
* **Hard-Locked Subpixel Purity:** `smoothmask` is held tightly to preserve un-diluted color chemistry and first-glow volume.
* **Symmetric Phase Alignment:** `s_sharp` acts as a horizontal-only phase filter to crush non-integer aspect scaling waves (e.g., PC Engine 256px) inside the gun pass, keeping vertical scanline trenches perfectly dark and deep.
* **WRGB Gate Alignment:** Modified phase windows absorb the horizontal spatial offset caused by the panel's physical White subpixel gate, releasing massive 3D optical pop.
* **High-Fidelity Grid Alignment:** Uses custom viewport mapping to eliminate standard down-sampling, locking rendering to full resolution for pixel-perfect triad alignment.
* **Insulated Pipeline Processing:** Chains the multi-pass layout to protect color chemistry profiles from mid-stream compression, ensuring high-contrast scanlines.


## 🎨 3D Color LUT Chemistry Mapping
The included 1024x32 color stripe files handle the non-linear analog color spaces of the original monitors:
* `astro-nanao-chemical.png` translates the high-vibrancy, wide-gamut arcade P22 phosphor saturation of the Nanao MS9 board.
* `trinitron-lut.png` secures the tight, clinical D65 reference broadcast colors of the Sony XBR chassis.
* **Pipeline Protection:** These textures are loaded natively inside `Pass 5 (LinearizePass)`. By eliminating vertical gun defocus row-bleeding and digital sharpness overshoots downstream, the raw RGB vectors are protected from reverse-voltage back-pressure, allowing the panel to display pure, uncompromised color saturation.

## 🚀 Milestone Release Notes: v33.0 (The 4K LCD Expansion)

This milestone release expands our Absolute Analog standard to native 4K LCD display panel substrates, completely decoupling our architecture from fixed-hardware limitations. 

### 📐 Unified Subpixel Switch Interface
To eliminate repository bloat and ensure plug-and-play simplicity for the community, v33.0 collapses our LCD deployment into a single, unified file structure for each profile. Physical subpixel layout tracking is managed directly inside the RetroArch overlay interface via a dynamic hardware pointer switch:

*   **Onscreen Parameter Entry:** Open the RetroArch **Quick Menu ➔ Shaders ➔ Shader Parameters** list and locate the **`Mask Layout`** slider variable.
*   **`0.00` ──► Standard RGB Base (Default File Initialization):** Calibrated for native pixel alignment on standard desktop gaming monitors and business laptop screens.
*   **`1.00` ──► Inverted BGR Base (One-Click Inversion):** Immediately swaps the driver's internal color indexing sequence to Blue-Green-Red. This completely neutralizes green/magenta color-fringing and restores needle-sharp text definition when running on living room 4K VA/IPS televisions or Gigabyte ADS monitors.

### 🛠️ Technical Change Control Matrix
The v33.0 LCD variant profiles take our frozen v32.0 core engine snapshots and implement a precise 3-point mathematical refactor to flatten the pipeline for flat linear matrices, ensuring absolute zero visual regression:

*   **High-Fidelity Grid Alignment:** By stripping away the horizontal phase cushions (`smoothmask = 0.000000`), the mask engine completely deactivates the light-bleeding layers originally needed to mask the physical white subpixel gate of WOLED glass. The emulated slot-mask dot-triads and aperture grille wires resolve with raw, uncompressed pixel bite.
*   **Symmetric Gun Precision:** Adjusting the subtractive horizontal sharpness boundaries stabilizes edge tracking within the horizontal signal plane. Non-integer horizontal aspect stretching waves are cleanly erased without introducing any vertical scanline row bleeding or contrast loss.
*   **Chassis Geometry & Decay Preservation:** All underlying CRT profile characteristics—including the active dual-axis spherical tube warping for the Nanao honeycomb profile, the horizontal-only cylindrical bow for the Trinitron wire mesh profile, and the custom high-retention phosphor persistence decay loops—remain entirely untouched.

### ⚠️ Physical Backlight Contrast Note
While the frame buffer output generated by our v33.0 code is mathematically pristine across all profiles, users loading these presets on an LCD panel must account for physical hardware limitations. Because standard LCD monitors and televisions rely on an LED backlight matrix, liquid crystal substrates cannot physically duplicate the absolute zero-volt ink-black scanline trenches of a self-emissive OLED panel. The emulated triads will resolve with perfect geometric focus, but final 3D optical contrast depth is ultimately bounded by the user's specific backlight hardware capabilities.
